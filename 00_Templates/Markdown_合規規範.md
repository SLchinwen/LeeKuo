# Markdown 合規規範

- **用途：** 避免 markdownlint 警告與格式不一致
- **適用：** 所有 `.md` 文件

## 1. 標題與清單

- 標題前後需保留一行空白（對應 MD022）
- 清單前後需保留一行空白（對應 MD032）

## 2. 表格格式

- 表格欄位兩側需保留空白（對應 MD060）
- 正確格式範例：

| 欄位 | 說明 |
| --- | --- |
| A | B |

## 3. 其他建議

- 盡量使用 ASCII
- 避免多餘空行與尾端空白

## 4. 標題使用規則

- 不要用粗體當標題（對應 MD036）
- 避免重複標題文字（對應 MD024）
- 建議用「粗體＋冒號」作為小標籤，而不是標題
