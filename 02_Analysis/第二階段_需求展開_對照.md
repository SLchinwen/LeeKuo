## 2. 欄位與規則
- **need_report / need_photo**：控制回報與拍照入口
- **photo_status**：Pending / Completed / NotRequired
- **crane_tonnage / parking_zone_group**：吊車彙總與看板

## 3. 必要驗收條件
- need_photo=否 → 回報不可被照片阻擋
- need_photo=是 → 回報完成但 photo_status=Pending
- 文件產出需記錄篩選條件與產出時間
- PhotoPending 可核定/請款
- 退回後可重新回報
